## React Typescript Base Template

> Simple get started react template

![](Application.png)

## Installation

OS X Windows & Linux:

```sh
yarn or npm install
```

## Execute

OS X Windows & Linux:

```sh
yarn start or npm start
```

## Main Technology

This project are developed using ReactJS at version 16.13.1
